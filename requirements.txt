    DB: PostgreSQL
    ORM/migrations/seeding: Prisma
    Auth: Cookie-based session auth (httpOnly, secure flags in prod) + CSRF protection
    Session storage: Postgres-backed sessions (avoid Redis to reduce cost/ops)
    Reverse proxy: Nginx (already used) and keep Docker Compose dev workflow

    Security headers: helmet
    Rate limiting on auth + chatbot endpoints
    Images: initially store image URLs in DB (no upload pipeline yet)
